# VHDL support for VSCode
VHDL support based on [https://github.com/yangsu/sublime-vhdl](https://github.com/yangsu/sublime-vhdl) SumblieText package.

## Features
### Done
- Syntax highlighting for `.vhd` `.vhdl` `vho` files

### In progress
- Simple Snippets
